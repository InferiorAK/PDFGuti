- Specillay Optimized for Desktop
- Also supported in Termux

## Commands
- ` git clone https://github.com/InferiorAK/PDFGuti `
- ` cd PDFGuti `
- ` pip3 install -r requirements.txt `
- ` python3 ppd.py `
